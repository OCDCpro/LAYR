# The LAYR hardware kit

##  Description
The LAYR hardware kit will be handed out to all teams. Each team gets one hardware kit.

## Content in the kit

### Devices:
* 1 x NFC RC522 Reader, SPI
* 1 x Memorychip AT205010B on Pinheader board, SPI
* 1 x Keycard, Java, pre-programmed

### Power:
* 1 x DC-DC stepup
* 1 x Relais 12V
* 1 x Powersupply board USB-C 

### Actors and Sensors:
* 1 x Doorlock 12V
* 5 x LEDs + resistors
* 2 x Pushbuttons

### Notes:
- The complete box is a gift.
- No return.
- Do want you want with it.
